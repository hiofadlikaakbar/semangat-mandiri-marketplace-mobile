# Semangat Mandiri Marketplace

Semangat Mandiri Marketplace merupakan aplikasi marketplace berbasis Flutter yang digunakan untuk melakukan pembelian produk sembako secara digital. Aplikasi ini telah terintegrasi dengan Firebase Authentication, Cloud Firestore, serta Deep Link yang terhubung dengan aplikasi **Semangat Mandiri Wallet** sebagai media pembayaran.

Pengguna dapat melakukan registrasi, login, mencari produk, melihat daftar produk, menambahkan produk ke keranjang, melakukan checkout, dan menyelesaikan pembayaran melalui aplikasi Wallet menggunakan verifikasi PIN.

---

## Fitur

* Registrasi akun (Sign Up)
* Login akun (Sign In)
* Logout akun
* Menampilkan daftar produk
* Mencari produk
* Menambahkan produk ke cart
* Menghapus produk dari cart
* Checkout produk
* Pembayaran menggunakan Deep Link ke aplikasi Semangat Mandiri Wallet
* Konfirmasi pembayaran menggunakan PIN
* Melihat riwayat transaksi

---

## Teknologi yang Digunakan

* Flutter
* Dart
* Firebase Authentication
* Cloud Firestore
* Firebase Core
* Deep Link

---

## Alur Aplikasi

1. Pengguna melakukan registrasi atau login.
2. Pengguna melihat daftar produk yang tersedia.
3. Pengguna dapat mencari produk berdasarkan nama.
4. Produk yang dipilih ditambahkan ke dalam cart.
5. Pengguna melakukan checkout.
6. Marketplace akan membuka aplikasi Semangat Mandiri Wallet menggunakan Deep Link.
7. Pengguna memasukkan PIN pada aplikasi Wallet untuk mengonfirmasi pembayaran.
8. Setelah pembayaran berhasil, transaksi akan disimpan dan ditampilkan pada halaman riwayat transaksi.
